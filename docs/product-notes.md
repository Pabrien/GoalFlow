# Product Notes

## Problem

Many to-do apps focus on reminders only. GoalFlow is designed around goals first, so users can understand why each task matters and keep motivation high.

## Differentiators

- Goal setting is built into the workflow
- Saved tasks can be reused and dragged into a calendar
- Reminders can be set per task
- Progress is visualized with charts and goal reports
- The UI stays simple and avoids unnecessary choices

## Current Screens

- Home: progress dashboard with completion rates, charts, and goal report
- Goals: goal list and goal creation
- Schedule: saved tasks plus weekly/monthly calendar
- Tasks: saved task list and task creation
- Today: today's scheduled tasks

## Product Priorities

- Persist data locally first, before adding accounts or a database
- Treat mobile as the primary use case
- Keep first-run onboarding focused on one goal, one task, and today's action
- Make empty states encouraging and action-oriented
- Emphasize the loop: goal -> today's task -> visible progress -> continued motivation
